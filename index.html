Design notes & export

Each logo variation is production-ready SVG. For Illustrator or CorelDRAW, export as SVG and open directly. For Figma, paste the SVG code as a vector frame. For Canva, use the PNG export.

Adobe Illustrator
File → Open → .svg — all vectors are editable paths
Figma
Copy SVG code → Edit → Paste as vector in canvas
CorelDRAW
Import → SVG — objects auto-separate into layers
Canva
Upload SVG → Uploads tab → drag onto canvas
